Assignment Summaries

Assignment 1: Created a simple user input program that provided some information about their name. We also did some manipulations with python lists. For the written portion, we wrote some pseudocode and did some base conversions.
Assignment 2: Filled in the skeleton code to create a trajectory simulator. Also answered some questions about Big-Oh for the written portion.
Assignment 3: Filled in the skeleton code to create a percolation module. Wrote some recursive functions and did a pseudo-implementation of a set for the written portion.
Assignment 4: Filled in the skeleton code for the teacher-assginment-student program. This was a basic introduction to OOP.
Midterm: Created a photo effects program, where users could select the filter they wanted to apply to a given photo.
Assignment 5: Implemented a few ML methods, namely KNN (twice) and SVM. Printed some statistics about the dataset, then used seaborn to visualize relationships in the data.
